Log Sentinel - Instalacao Windows
=================================

Arquivos esperados neste pacote:
- sentinel.exe
- setup.bat
- sentinel.env.example

Passo a passo:
1. Extraia o pacote em uma pasta local, por exemplo:
   C:\Aiknow\acc_log_sentinel
2. Abra o Prompt de Comando como Administrador.
3. Entre na pasta do pacote.
4. Execute:
   setup.bat
5. Informe:
   - endpoint da API
   - token da API
   - intervalo de coleta
6. Confirme o status do servico ao final.

Comandos uteis:
- sentinel.exe status
- sentinel.exe run-once
- sentinel.exe stop
- sentinel.exe start

Pre-requisitos:
- acesso HTTPS ao backend central
- token valido
- permissao de administrador para instalar servico Windows
